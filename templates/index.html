ℹ Over ASTRO-impact

ASTRO-impact is een interactieve asteroïde-impact simulator gebouwd met Python/Flask.


Databronnen

• NASA NeoWs API – Near-Earth Object data

• REST Countries API – oppervlakte & landcoördinaten

• World Bank API – actuele bevolkingscijfers (2024)

• CartoDB – kaart tegels


Formules

• Kinetische energie: E = ½mv² (dichtheid steen 3000 kg/m³)

• Impactzones: geschaald vanuit Hiroshima (63 TJ, 1.4 km lethal radius)

• Magnitude: M = (log₁₀(E) − 4.8) / 1.5
